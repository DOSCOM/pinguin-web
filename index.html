Pinguin dinus repository

Server Repository Pinguin merupakan sebuah mirror kumpulan free /open source software yang dipersembahkan oleh Universitas Dian Nuswantoro , Jawa Tengah Open Source Center (JOSC), dan Dinus Open Source Community (DOSCOM) untuk para pengguna dan pengembang free /open source software di Indonesia. Kami berharap dengan adanya repository ini, akan sangat bermanfaat bagi perkembangan dunia free / open source software di lingkungan Universitas Dian Nuswantoro pada khususnya dan Indonesia pada umumnya.

TeaLinux OS

LinuxOS adalah distro Linux turunan Ubuntu yang dikembangkan oleh Dinus Open Source Community (DOSCOM) yang berorientasi pemrograman. Dengan menghadirkan filosofi, “Nikmatnya sebuah racikan”, TeaLinuxOS dikembangkan secara terbuka dan bersama-sama untuk menghasilkan distro Linux pemrograman yang dikhususkan untuk dunia pendidikan.
